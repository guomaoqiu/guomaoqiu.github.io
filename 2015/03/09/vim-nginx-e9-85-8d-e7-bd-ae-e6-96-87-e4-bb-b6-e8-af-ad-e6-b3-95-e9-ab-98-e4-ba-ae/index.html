Vim Nginx配置文件语法高亮

2015-03-09
  • Shell

我们在编辑配置nginx的配置文件时，由于他没有高亮的功能，但是nginx官方是支持这个功能的；要想在编辑配置nginx配置文件的时候高亮语法以降低配置的错误发生率可移执行这个小脚本而到达目的：

#!/bin/bash
mkdir -p ~/.vim/syntax && cd ~/.vim/syntax
wget http://www.vim.org/scripts/download\_script.php?src\_id=14376 -O nginx.vim >/dev/null
echo “au BufRead,BufNewFile /usr/local/webserver/nginx/conf/* set ft=nginx” > ~/.vim/filetype.vim

#其中路径/usr/local/webserver/nginx/conf/*为你的nginx.conf文件路径
